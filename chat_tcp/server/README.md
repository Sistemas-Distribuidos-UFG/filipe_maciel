# Aplicação
    * go build .

# Servidor

## Executar o build
    * ./chat_tcp


## Cliente
    * telnet localhost 8888

## Comandos 
### Criar usuário
    * /nick nome_do_usuario

### Entrar na sala
    * /join #sala 
### Enviar mensagem 
    * /msg mensagem
### Listar salas 
    * /rooms

### Sair da sala
    * /quit
